# DataCamp Finance

## Installation

You'll need the [TA-Lib](https://ta-lib.org) C library:

```sh
# mac
brew install ta-lib

# linux
wget https://github.com/TA-Lib/ta-lib/releases/download/v0.4.0/ta-lib-0.4.0-src.tar.gz
tar -xvf ta-lib-0.4.0-src.tar.gz
cd ta-lib
./configure --prefix=/usr
make
sudo make install
```
